# gh-pages

Simplest way to do this is with vue-cli-plugin-gh-pages (opens new window)

# 1. Create Repository

https://github.com/robertleroy/PROJECT_NAME


# 2. Commit project

git init
git add .
git commit -m 'first commit'
git remote add origin https://github.com/robertleroy/PROJECT_NAME.git
git push -u origin master

# 3. Add & run plugin

vue add gh-pages
npm run gh-pages

Thats it.
